# 2048 Game 🎯

중독성 강한 숫자 퍼즐 게임! 타일을 합쳐서 2048을 만들어보세요.

## 게임 방법

### 조작
- **방향키 (↑ ↓ ← →)**: 타일 이동
- **모바일**: 스와이프로 조작
- **되돌리기**: 이전 단계로 돌아가기 (1회만 가능)

### 게임 규칙
1. 방향키를 누르면 모든 타일이 해당 방향으로 이동합니다
2. 같은 숫자를 가진 타일이 만나면 **합쳐집니다**
3. 2 + 2 = 4, 4 + 4 = 8, 8 + 8 = 16...
4. **2048** 타일을 만들면 승리!
5. 더 이상 움직일 수 없으면 게임 오버

### 전략 팁
💡 **초보자**:
- 큰 숫자를 한 구석에 모으세요 (보통 아래-오른쪽)
- 한 방향을 주로 사용하고 가끔 다른 방향 사용
- 급하게 서두르지 마세요!

💡 **고급**:
- 항상 가장 큰 타일을 구석에 유지
- 두 번째로 큰 타일을 옆에 배치
- 하나의 행/열을 가득 채우지 않기
- 계획적으로 타일 배치하기

## 주요 기능

### 🎮 게임 플레이
- **부드러운 애니메이션**: 타일 이동과 합치기 효과
- **되돌리기**: 실수했을 때 1단계 되돌리기
- **승리 후 계속**: 2048 달성 후에도 더 높은 점수 도전
- **효과음**: 이동, 합치기 사운드

### 📊 통계
- **현재 게임**:
  - 이동 횟수 추적
  - 최대 타일 표시
- **전체 통계**:
  - 총 게임 수
  - 승리 횟수
  - 최고 점수 기록

### 🏆 도전 과제
- **🎯 첫 승리**: 2048 타일 만들기
- **⚡ 스피드 데몬**: 100회 이하 이동으로 승리
- **💎 하이 롤러**: 4096 타일 만들기
- **🛡️ 생존왕**: 500회 이상 이동하기

### 🎨 디자인 특징
- **그라디언트 배경**: 보라색 계열의 아름다운 배경
- **타일 색상**: 값에 따라 달라지는 색상 (2~8192)
- **팝 애니메이션**: 타일이 합쳐질 때 튀는 효과
- **반응형 디자인**: 모바일/태블릿/데스크톱 완벽 지원

## 기술 구현

### 게임 엔진
- **그리드 관리**: 4x4 2차원 배열
- **이동 알고리즘**: 회전 + 왼쪽 이동 + 역회전
- **합치기 로직**: 인접한 같은 숫자 병합
- **승리/패배 감지**: 자동 체크

### 애니메이션
- **CSS Transitions**: 부드러운 타일 이동
- **CSS Animations**: 팝, 나타나기 효과
- **Keyframes**: 도전과제 해금 애니메이션

### 데이터 저장
- **LocalStorage**: 최고 점수, 통계, 도전과제
- **상태 관리**: 되돌리기를 위한 히스토리
- **JSON 직렬화**: 데이터 저장 및 복원

## 게임 기록

### 세계 최고 점수
- 공식 2048 최고 점수: **131,072 타일** 도달 가능
- 평균 승리 이동 횟수: 약 250-350회
- 빠른 승리 기록: 100회 이하

### 전략 가이드
1. **구석 전략**: 가장 효과적
   - 큰 타일을 한 구석에 고정
   - 옆 두 방향만 주로 사용

2. **스네이크 전략**: 고급
   - 타일을 뱀 모양으로 배치
   - 큰 숫자부터 작은 숫자 순서

3. **방어적 플레이**
   - 빈 칸을 최대한 많이 유지
   - 급하게 움직이지 않기
