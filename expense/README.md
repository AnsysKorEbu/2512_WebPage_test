# Expense Tracker 💰

개인 재정 관리를 위한 스마트 가계부 애플리케이션입니다.

## 주요 기능

### 💸 거래 관리
- **수입/지출 추가**: 금액, 날짜, 카테고리, 메모 입력
- **카테고리 분류**:
  - 지출: 식비, 교통, 쇼핑, 주거, 의료, 문화, 기타
  - 수입: 급여, 부수입, 투자
- **거래 내역 조회**: 월별, 유형별 필터링
- **거래 삭제**: 잘못 입력한 거래 삭제 기능

### 📊 시각화 & 통계
- **요약 카드**: 월별 총 수입, 지출, 잔액 표시
- **파이 차트**: 카테고리별 지출 비율 시각화
- **카테고리 범례**: 각 카테고리별 금액 상세 표시

### 🎯 예산 관리
- **월별 예산 설정**: 각 월마다 다른 예산 설정 가능
- **진행률 표시**: 예산 사용률을 프로그레스 바로 시각화
- **경고 시스템**:
  - 80% 이상: 노란색 경고
  - 100% 이상: 빨간색 초과 경고

### 📥 데이터 관리
- **자동 저장**: 모든 데이터는 브라우저 LocalStorage에 자동 저장
- **월별 탐색**: 이전/다음 달로 쉽게 이동
- **JSON 내보내기**: 데이터를 JSON 파일로 백업

## 사용 방법

1. **거래 추가**
   - 수입/지출 유형 선택
   - 금액, 날짜, 카테고리 입력
   - 메모 추가 (선택사항)
   - "추가하기" 버튼 클릭

2. **예산 설정**
   - 예산 카드의 ⚙️ 버튼 클릭
   - 월 예산 금액 입력
   - 저장

3. **거래 조회**
   - 월 선택기로 원하는 월 선택
   - 전체/수입/지출 필터로 보기 전환
   - 거래 항목의 × 버튼으로 삭제

4. **데이터 백업**
   - "데이터 내보내기" 버튼 클릭
   - JSON 파일이 자동 다운로드

## 데이터 구조

### Transaction (거래)
```javascript
{
  id: 1234567890,
  type: 'expense' | 'income',
  amount: 50000,
  date: '2025-12-03',
  category: '식비',
  description: '저녁 회식',
  createdAt: '2025-12-03T10:30:00.000Z'
}
```

### Budget (예산)
```javascript
{
  '2025-12': 1000000,  // 2025년 12월 예산
  '2026-01': 1200000   // 2026년 1월 예산
}
```

## 기술 스택

- **HTML5**: 구조
- **CSS3**: 스타일링 (그라디언트, 애니메이션)
- **Vanilla JavaScript**: 로직 (ES6 Modules)
- **Canvas API**: 차트 그리기
- **LocalStorage**: 데이터 저장
