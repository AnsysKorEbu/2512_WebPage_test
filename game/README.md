# Snake Game 🐍

클래식 뱀 게임의 현대적인 재해석! 네온 스타일의 레트로 게임입니다.

## 게임 방법

### 조작 키
- **방향키 (↑ ↓ ← →)**: 뱀 이동
- **SPACE**: 일시정지/재개
- **터치/스와이프**: 모바일 조작 지원

### 게임 규칙
1. 뱀을 조종하여 사과를 먹습니다
2. 사과를 먹을수록 뱀이 길어집니다
3. 벽이나 자기 몸에 부딪히면 게임 오버!
4. 5개의 사과를 먹을 때마다 레벨 업 (속도 증가)

### 아이템
- 🍎 **빨간 사과**: +10점
- 🌟 **황금 사과**: +50점 (10% 확률로 출현)

## 주요 기능

### 🎮 게임 플레이
- **부드러운 움직임**: 60 FPS 렌더링
- **레벨 시스템**: 난이도가 점진적으로 증가
- **점수 시스템**: 일반/황금 사과로 점수 획득
- **효과음**: 사과 먹기, 레벨업, 게임 오버 사운드

### 📊 통계 & 기록
- **실시간 점수**: 현재 점수, 레벨 표시
- **최고 점수**: LocalStorage에 자동 저장
- **리더보드**: 상위 10개 기록 저장
- **게임 통계**: 먹은 사과 수, 생존 시간 기록

### 🎨 시각 효과
- **네온 글로우**: 뱀과 사과의 발광 효과
- **그리드 시스템**: 게임 필드 그리드 표시 (옵션)
- **부드러운 애니메이션**: CSS 애니메이션과 트랜지션
- **레트로 UI**: Press Start 2P 폰트 사용

### 🔧 치트 모드
- **🛡️ 무적 모드**: 벽 통과 가능, 자기 몸 충돌 무시
- **🐌 슬로우 모션**: 속도 50% 감소
- **⚡ 속도 증가**: 속도 2배 증가
- **주의**: 치트 사용 시 점수 저장 안 됨!

### ⚙️ 설정
- **효과음 ON/OFF**: 사운드 켜기/끄기
- **그리드 표시**: 게임 필드 그리드 토글
- **기록 초기화**: 모든 저장된 데이터 삭제

## 기술 스택

- **HTML5 Canvas**: 게임 렌더링
- **Vanilla JavaScript**: 게임 로직 (ES6 Modules)
- **CSS3**: 네온 효과, 애니메이션
- **Web Audio API**: 효과음 생성
- **LocalStorage**: 점수 및 설정 저장
