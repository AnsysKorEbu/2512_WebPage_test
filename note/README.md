# Note App

간단한 Notion 스타일의 노트 애플리케이션입니다.

## 기능

- 📝 여러 페이지 생성 및 편집
- 💾 로컬스토리지 자동 저장
- 📂 JSON 파일로 내보내기/가져오기
- 🎨 페이지 아이콘 변경
- ⌨️ 키보드 단축키 (Enter로 새 블록, Backspace로 병합)

## 사용 방법

1. `index.html`을 브라우저에서 엽니다
2. 왼쪽 사이드바에서 "페이지 추가"를 클릭하여 새 페이지를 만듭니다
3. 제목과 내용을 입력합니다
4. 모든 변경 사항은 자동으로 저장됩니다
5. "저장하기" 버튼으로 JSON 파일로 백업할 수 있습니다
6. "불러오기" 버튼으로 이전에 저장한 노트를 복원할 수 있습니다

## 파일 구조

```
note/
├── index.html      # 메인 HTML
├── css/
│   └── style.css   # 스타일시트
└── js/
    ├── app.js      # 메인 앱 로직
    └── storage.js  # 데이터 저장/로드
```

## 저장 위치

- **로컬스토리지**: 브라우저에 자동 저장 (키: `notion_clone_data`)
- **파일 백업**: "저장하기" 버튼으로 `storage/` 폴더에 JSON 파일로 백업 가능
