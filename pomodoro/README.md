# Pomodoro Focus Timer 🍅

뽀모도로 기법을 활용한 생산성 향상 타이머 애플리케이션입니다.

## 기능

### ⏱️ 타이머
- **집중 시간**: 기본 25분 (설정 가능)
- **짧은 휴식**: 기본 5분 (설정 가능)
- **긴 휴식**: 기본 15분 (설정 가능)
- 원형 프로그레스 바로 시각화
- 일시정지/재시작 기능

### ✅ 작업 관리
- 오늘의 작업 목록 추가/삭제
- 작업 완료 체크
- 작업별 뽀모도로 카운트 (더블클릭으로 증가)
- 완료된 작업 일괄 삭제

### 📊 통계
- 오늘 완료한 뽀모도로 개수
- 총 완료한 뽀모도로 개수
- 자정마다 자동 리셋

### ⚙️ 설정
- 타이머 시간 커스터마이징
- 자동 휴식 시작 옵션
- 알림음 활성화/비활성화
- 브라우저 알림 지원

## 사용 방법

1. `index.html`을 브라우저에서 엽니다
2. 작업 목록에 오늘 할 일을 추가합니다
3. 타이머를 시작하고 25분간 집중합니다
4. 타이머가 끝나면 5분간 휴식합니다
5. 4번의 집중 후 긴 휴식을 취합니다

## 뽀모도로 기법이란?

뽀모도로 기법은 시간 관리 방법론으로:
- 25분 집중 작업
- 5분 휴식
- 4회 반복 후 15-30분 긴 휴식

이를 통해 집중력을 유지하고 번아웃을 방지합니다.

## 키보드 단축키

- `Space`: 타이머 시작/일시정지
- `Escape`: 설정 모달 닫기
- `Enter`: 작업 추가 (입력 필드에서)

## 데이터 저장

모든 데이터는 브라우저의 LocalStorage에 저장됩니다:
- `pomodoroSettings`: 타이머 설정
- `pomodoroTasks`: 작업 목록
- `pomodoroStats`: 통계 데이터

## 기술 스택

- HTML5
- CSS3 (그라디언트, 애니메이션)
- Vanilla JavaScript (ES6 Modules)
- Web Audio API (알림음)
- Notification API (브라우저 알림)
- LocalStorage (데이터 저장)
